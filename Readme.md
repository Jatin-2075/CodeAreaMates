
# College Mate

College Mate is a web platform designed to help students connect, collaborate, and participate in college activities in one place.

Instead of scattered WhatsApp groups and confusion, everything related to college collaboration lives in a single organized system.

## Key Features

- User authentication and profiles
- Explore and join events
- Send participation proposals
- View teammates (mates)
- Post and read event stories
- Track achievements and past events

## Tech Stack

- **Frontend:** React, TypeScript
- **Backend:** Django
- **Database:** (PostgreSQL/MySQL - specify as needed)

## Getting Started

### Prerequisites
- Node.js (v14+)
- Python (v3.8+)
- pip

### Installation

**Backend:**
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

**Frontend:**
```bash
npm install
npm start
```
